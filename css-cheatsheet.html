CSS Notes (Beginner to Advanced)

1. What is CSS?

CSS (Cascading Style Sheets) is used to style and layout HTML elements.

2. CSS Syntax

selector {
  property: value;
}

3. Ways to Add CSS

Inline:   <h1 style="color:red;">Hello</h1>
Internal: <style> h1 { color: red; } </style>
External: <link rel="stylesheet" href="style.css">

4. Selectors

/* Basic */
p { color: blue; }
#idName { color: green; }
.className { color: red; }

/* Grouping */
h1, h2, p { margin: 0; }

/* Descendant */
div p { color: purple; }

/* Child */
div > p { color: orange; }

/* Pseudo-class */
a:hover { color: red; }
li:first-child { font-weight: bold; }

/* Pseudo-element */
p::first-line { color: blue; }
p::before { content: "â˜… "; }

5. Colors & Units

/* Color values */
color: red;
color: #ff0000;
color: rgb(255,0,0);
color: rgba(255,0,0,0.5);
color: hsl(0,100%,50%);

/* Units */
px (pixels)
em (relative to parent)
rem (relative to root)
% (percentage)
vw, vh (viewport width/height)

6. Box Model

div {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
  box-sizing: border-box;
}

7. Text & Fonts

p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  font-style: italic;
  text-align: center;
  text-decoration: underline;
  line-height: 1.5;
  letter-spacing: 2px;
}

8. Backgrounds

body {
  background-color: lightblue;
  background-image: url("bg.jpg");
  background-repeat: no-repeat;
  background-size: cover;
  background-position: center;
  background-attachment: fixed;
}

9. Borders & Shadows

div {
  border: 2px solid red;
  border-radius: 10px;
  box-shadow: 2px 2px 10px gray;
  text-shadow: 1px 1px 2px black;
}

10. Display & Positioning

/* Display */
div { display: block; }
span { display: inline; }
.flex { display: flex; }
.grid { display: grid; }

/* Position */
.relative { position: relative; top: 10px; left: 20px; }
.absolute { position: absolute; top: 0; right: 0; }
.fixed { position: fixed; bottom: 0; }
.sticky { position: sticky; top: 0; }

11. Flexbox

.container {
  display: flex;
  justify-content: center;   /* flex-start, flex-end, space-between */
  align-items: center;       /* stretch, flex-start, flex-end */
  flex-wrap: wrap;
}
.item {
  flex: 1;    /* grow */
}

12. CSS Grid

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 10px;
}
.item {
  grid-column: span 2;
  grid-row: 1 / 3;
}

13. Transitions

button {
  background: blue;
  color: white;
  transition: background 0.3s ease, transform 0.3s;
}
button:hover {
  background: red;
  transform: scale(1.1);
}

14. Animations

@keyframes move {
  0% { transform: translateX(0); }
  50% { transform: translateX(100px); }
  100% { transform: translateX(0); }
}
div {
  animation: move 3s infinite;
}

15. Media Queries (Responsive Design)

@media (max-width: 600px) {
  body { background: lightgray; }
  p { font-size: 14px; }
}

16. Variables (Custom Properties)

:root {
  --main-color: blue;
  --padding: 10px;
}
div {
  color: var(--main-color);
  padding: var(--padding);
}

17. Advanced Selectors

/* Attribute selectors */
input[type="text"] { border: 1px solid gray; }
a[target="_blank"] { color: red; }

/* nth-child */
li:nth-child(odd) { background: #eee; }
li:nth-child(2) { color: blue; }

18. Z-Index

div {
  position: absolute;
  z-index: 10; /* Higher value appears on top */
}

19. Clipping & Masking

div {
  clip-path: circle(50%);
  -webkit-mask-image: url(mask.png);
}

20. Filters & Blend Modes

img {
  filter: blur(5px) brightness(0.8) contrast(120%);
  mix-blend-mode: multiply;

24. CSS Responsive Units

/* Default style */
body {
  background-color: lightblue;
  font-size: 18px;
}

/* For tablets and smaller (max-width: 768px) */
@media (max-width: 768px) {
  body {
    background-color: lightgreen;
    font-size: 16px;
  }
}

/* For mobile screens (max-width: 480px) */
@media (max-width: 480px) {
  body {
    background-color: lightcoral;
    font-size: 14px;
  }
}

25. CSS Best Practices

  • Use external CSS files for maintainability.
  • Follow a naming convention (BEM recommended).
  • Prefer flexbox & grid over floats for layout.
  • Use CSS variables for consistency.
  • Keep responsive design in mind from the start.
