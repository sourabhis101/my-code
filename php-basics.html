Php — Basics to Advanced (Cheat‑sheet & Examples)

Readable, copy-ready examples to learn php from first steps to advanced patterns.

Setup & Environment

Install PHP (recommended latest stable). Use PHP built-in server for development: php -S localhost:8000. Use Xdebug for debugging and PHP-CS-Fixer / phpcs for linting.

// check version
php -v

// run built-in server
php -S localhost:8000 -t public

Hello World

<?php

echo "Hello, PHP!\n";

?>

Save as index.php and visit http://localhost:8000.

<?php
// variables
$int = 10;

$bool = true;

define('APP_NAME', 'MyApp');

Control Flow

  echo "positive";

  echo "negative";

  case 'a': break;

Functions

<?php
// simple function
function greet(string $name = 'Guest'): string {
  return "Hello, $name";
}

// variadic
function sumAll(int ...$nums): int {
  return array_sum($nums);
}

// closures
$adder = fn($a,$b) => $a + $b; // arrow fn (php 7.4+)

?>

OOP: Classes & Objects

<?php
class User {
  private string $name;
  public function __construct(string $name) { $this->name = $name; }
  public function getName(): string { return $this->name; }
  public static function makeGuest(): self { return new self('Guest'); }
}

// inheritance
class Admin extends User {
  public function isAdmin(): bool { return true; }
}

// interfaces & traits
interface Logger { public function log(string $msg): void; }
trait Timestamped { public function now(): string { return date('c'); } }

?>

Use typed properties (PHP 7.4+). Favor composition over deep inheritance.

Exceptions & Errors

<?php
try {
  // something
} catch (\Exception $e) {
  error_log($e->getMessage());
  throw $e; // or handle
} finally {
  // cleanup
}

// custom exception
class MyError extends \Exception {}

?>

Arrays & Iterables

<?php
// associative array
$user = ['id'=>1, 'name'=>'Alice'];

// indexed
$nums = [1,2,3];

// array functions
array_map(fn($v) => $v * 2, $nums);
array_filter($nums, fn($v) => $v > 1);

// traversing
foreach ($user as $k => $v) { echo "$k: $v\n"; }

?>

Databases (PDO)

<?php
$dsn = 'mysql:host=127.0.0.1;dbname=testdb;charset=utf8mb4';
$pdo = new PDO($dsn, $user, $pass, [PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION]);

// prepared statement
$stmt = $pdo->prepare('SELECT id, name FROM users WHERE id = :id');
$stmt->execute(['id' => $id]);
$row = $stmt->fetch(PDO::FETCH_ASSOC);

// transactions
$pdo->beginTransaction();
try {
  // ...
  $pdo->commit();
} catch (Exception $e) {
  $pdo->rollBack();
  throw $e;
}

?>

HTTP & Web APIs

<?php
// get input
$name = $_GET['name'] ?? 'guest';

// JSON response
header('Content-Type: application/json');
echo json_encode(['name' => $name]);

// simple router (very small)
$path = $_SERVER['REQUEST_URI'];
switch ($path) {
  case '/': include 'views/home.php'; break;
  case '/api': include 'api.php'; break;
  default: http_response_code(404); echo 'Not found';
}

?>

For production APIs consider frameworks like Laravel, Symfony, or micro frameworks like Slim/Flight.

Security Best Practices

  • Always use prepared statements to avoid SQL injection.
  • Escape output (use htmlspecialchars) to prevent XSS.
  • Validate and sanitize user input.
  • Store password hashes using password_hash and verify with password_verify.
  • Use HTTPS and set proper headers (HSTS, Content-Security-Policy).
  • Limit file uploads and validate file types.

Tips & Best Practices

  • Use typed properties and strict types: declare(strict_types=1);.
  • Follow PSR standards (PSR-12 coding style, PSR-4 autoloading).
  • Keep controllers thin; put logic in services.
  • Use environment variables for config (Dotenv).
  • Run static analysis (PHPStan, Psalm).
