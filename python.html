1. What is Python & Why Learn It

Python is a high-level, interpreted language known for simplicity and readability. Used in web development, data analysis, machine learning, automation, scripting, and more.

  • Beginner-friendly syntax
  • Large standard library & third-party ecosystem
  • Cross-platform: Windows, Linux, macOS

2. Setup & Tools

Install from python.org or use system package managers. Verify with python --version / pip --version. Recommended editors: VS Code, PyCharm, Jupyter Notebook. Use virtual environments via python -m venv myenv.

19. Fibonacci Examples & Application

def fib(n):
    a,b = 0,1
    while a < n:
        print(a, end=' ')
        a,b = b, a+b
    print()

# fibonacci list
def fib_list(n):
    a,b = 0,1
    seq = []
    while a < n:
        seq.append(a)
        a,b = b, a+b
    return seq

Example application: Fibonacci retracement levels in trading (38.2%, 50%, 61.8%, 78.6%) calculated using ratios of sequence values. (See PDF for trading example.)

20. OOP Fundamentals

Classes, objects, attributes, methods, inheritance, encapsulation (private attributes using __), polymorphism, abstraction (abstract base classes).

# Class & object
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def info(self):
        return f"{self.name} is {self.age}"

s1 = Student("Harry", 21)
print(s1.info())

# Inheritance
class Animal:
    def speak(self):
        return "Animal sound"
class Dog(Animal):
    def speak(self):
        return "Bark"
print(Dog().speak())

# Encapsulation (convention)
class Box:
    def __init__(self):
        self._hidden = 42  # protected by convention
b = Box()
print(b._hidden)

# Polymorphism
def make_sound(animal):
    print(animal.speak())
make_sound(Dog())

# Class & static methods
class Util:
    rate = 1.18
    @classmethod
    def with_rate(cls, amount):
        return amount * cls.rate
    @staticmethod
    def add(a, b):
        return a + b

print(Util.with_rate(100))
print(Util.add(2, 3))

21. Decorators

def decorator(fn):
    def wrapper(*a, **k):
        print("Before")
        result = fn(*a, **k)
        print("After")
        return result
    return wrapper

@decorator
def hello(): print("Hello")

def count_up_to(n):
    i = 1
    while i <= n:
        yield i
        i += 1
for num in count_up_to(3): print(num)
